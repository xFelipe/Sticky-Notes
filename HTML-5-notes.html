Tags:

  • < p >: Trata-se de um paragrafo. Deve ser fechado.
  • < wbr >: Permite a quebra de texto no ponto onde é colocado.
  • id="segundo"> & shy: Permite a quebra de texto no ponto onde é colocado incluindo automaticamente um "-".
  • < nav >: Secao com links para outros sites ou paginacao(Bloco de links, rodape, menu). Da pra fazer listas de links e talz... Deve ser fechado.
  • < section >: Uma sessão de conteúdos.
  • < article >: Um bloco de conteúdo.
  • < aside >: Bara lateral/Sidebar. Deve ser fechado.
  • < footer >: E o rodape. O elemento final antes de feixar o HTML e pode conter um < nav >(não recomendado), logomarca... Deve ser fechado.
  • < a >: Serve para fazer links. Deve ser fechado.
  • < abbr >: Serve para colocar siglas e explicar o significado delas. Deve ser fechado.
  • < address >: Serve para colocar colocar endereço dentro. Deve ser fechado.
  • < blockquote > Tag que define citacoes LONGAS(trechos, textos). Deve ser fechado.
  • < cite > Tag que define citacoes CURTAS(referencia). Deve ser fechado.
  • < code > tag que define citacoes de CÓDIGOS. Deve ser fechado.
  • < div > e < span >: Definem estrutura de elementos. Podem trabalhar junto com o CSS. div e elemento de bloco e span é elemento de linha. Deve ser fechado.
  • < STRONG > Negrito. Deve ser fechado.
  • < EM > hitalico. Deve ser fechado.
  • < ol >: Lista nao ordenada(um monte de sombolos iguals, tipo "*" ou "-"). Suas opções são criadas com varios < li > em seu interior. Deve ser fechado.
  • < ul >: Lista ordenada(Usa numero ou letras para ordenar). Suas opções são criadas com varios < li > em seu interior. Deve ser fechado.

  • A tag DIV serve para agrupar um conteudo, organizar ele, fazer um cabecalho, menu e organizar este conteudo da forma que voce quiser. Nao e recomendado usar numeros no ID da DIV. As DIVs podem usar arquivos .CSS para se encrementar.
  • id: Identifica apenas um item.
  • class: Identifica vários itens.
  • < del >
  • < span style="text-decoration: underline";>
  • Tag de abreviação: < abbr> Ao passar o mouse por cima é possível ver o significado.

  • < span style="text-decoration: overline";>

  • & nbsp;: Espaçamento.

  • z-index: O item que tiver o maior valor z-index fica pra frente. É usado geralmente em itens com position:absolute
  • top: Coordenada inicial vertical.
  • left: Coordenada inicial horizontal.
  • width: largura do objeto
